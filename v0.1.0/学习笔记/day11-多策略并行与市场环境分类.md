# Day 11 — 多策略并行 + 市场环境分类器

> 2026-07-06 | Week 2 第四天

## 学了什么

### 两个赚钱的策略 ≠ 分散风险

这是多策略并行最大的认知陷阱。

假设你同时跑趋势跟踪和动量策略。回测数据显示两个策略都赚钱，看起来很好。但如果你计算它们的交易相关性——发现最近 20 笔交易的收益率相关性是 0.78。

这意味着：
- 趋势策略赚钱的时候，动量策略大概率也在赚钱
- 趋势策略亏钱的时候，动量策略大概率也在亏钱
- **你以为是两个策略，其实是一个策略穿了不同的马甲**

真正的多策略价值在于**对冲和互补**——一个策略的市场环境恰好是另一个的短板。

### 市场环境的本质：不同策略吃不同的行情

```
趋势跟踪 → 吃趋势市（连续的上涨或下跌）
动量因子 → 吃趋势市（和趋势策略高度重叠）
均值回归 → 吃震荡市（恰恰是趋势策略的亏损环境）
资金费率套利 → 吃任何行情（只要费率够高）
```

这就是为什么环境分类器不是"加分项"而是"核心模块"——它决定了你该让哪个策略上场。

### 五类市场环境

| 环境 | 特征 | ADX | ATR 百分位 | 均线排列 |
|------|------|-----|-----------|---------|
| TRENDING_UP | 强上升趋势 | > 25 | 正常 | 价格 > MA50 > MA200 |
| TRENDING_DOWN | 强下降趋势 | > 25 | 正常 | 价格 < MA50 < MA200 |
| RANGING | 窄幅震荡 | < 20 | 低 | MA 缠绕 |
| VOLATILE | 高波动 | 任意 | > 80% | 任意 |
| TRANSITIONING | 过渡期 | 任意 | 任意 | 无明确信号 |

### 如何分类：多维打分

单靠 ADX 一条线是不够的。分类器用 6 个维度综合打分：

1. **ADX** (趋势强度) — 趋势市 +30分，震荡市 +25分
2. **ATR 百分位** (波动率) — 高波动 +35分，低波动加分给震荡
3. **均线排列** (方向确认) — MA50 > MA200 + 价格在上 = 上升趋势 +25分
4. **布林带宽度** (波动扩张/收缩) — 带宽扩张 → 波动 +20分
5. **趋势一致性** (方向连续) — 连续 10 根同向 → 加分
6. **外部信号** (恐惧贪婪、资金费率) → 微调

最后**最高分当选**，所有维度得分都 < 20 分 → 过渡期（观望）。

### 为什么用"多维打分"而不是"if-else 规则"？

if-else 规则（如 "ADX > 25 AND price > MA50 = 趋势"）的问题是边界太硬：
- ADX = 24.9 vs 25.1 → 完全不同的分类？不合理
- 每个指标都有噪声，单一规则容易误判

打分制让所有指标平等投票，边界更平滑。ADX 24.9 时趋势得分略低，但均线排列和趋势一致性可以补上。

### 环境 → 策略乘数

每种环境给每种策略类型不同的仓位乘数：

| 策略 \ 环境 | 趋势上涨 | 趋势下跌 | 震荡 | 高波动 | 过渡期 |
|------------|---------|---------|------|-------|-------|
| 趋势跟踪 | **1.0** | **1.0** | 0.3 | 0.5 | 0.0 |
| 动量 | **1.0** | 0.7 | 0.3 | 0.5 | 0.0 |
| 突破 | 0.8 | 0.6 | 0.5 | 0.4 | 0.0 |
| 均值回归 | 0.3 | 0.3 | **1.0** | 0.5 | 0.3 |
| 费率套利 | 1.0 | 1.0 | 1.0 | 0.5 | 0.5 |

逻辑：
- **趋势市** → 趋势/动量满仓，均值回归减到 0.3（趋势市做均值回归 = 逆势操作）
- **震荡市** → 均值回归满仓，趋势/动量减到 0.3（震荡市做趋势 = 反复假突破）
- **高波动** → 全部减仓，止损放宽（避免被噪音扫出）
- **过渡期** → 不开新仓，持有观望

### 相关性降权

即使两个策略都适合当前环境，如果它们相关性太高，也需要降权：

```
趋势策略 × 动量策略 相关性 = 0.78 → > 0.60 阈值
→ 两者仓位各打 0.7 * 0.78 = 0.546 折扣
```

不是禁用其中一个（它们逻辑不同，可能在不同时间点给出信号），而是降低仓位。

### 净敞口监控

```
多头总仓位 35% - 空头总仓位 5% = 净多头 30%
```

多空同时持仓 = 自然对冲，降低市场方向性风险。如果净敞口超过 60%，说明你实际上在赌方向，而不是在做多策略分散。

### 集中度限制

同一方向最多 2 个策略同时持仓。3 个策略全部做多 = 你不是在做多策略，你是在三重做多。

### 仓位调整链

```
PositionSizer(基础仓位)
  × RegimeClassifier(环境乘数)
  × 相关性降权
  × 集中度检查
  → RiskManager(最终否决)
```

每一步都可以让仓位变小，绝不让它变大。最终 RiskManager 有最终否决权。

   - 6 维综合打分：ADX + ATR 百分位 + 均线排列 + 布林带宽度 + 趋势一致性 + 外部信号
