# Day 4 学习笔记 — 动量因子 + 资金费率套利

## 一、动量因子

### 1.1 动量的本质

> 动量 = 过去 N 天的涨跌幅度。过去涨得多的，未来倾向于继续涨；过去跌得多的，未来倾向于继续跌。

这不是「追涨杀跌」——追涨杀跌是情绪驱动。动量策略是统计驱动：历史数据显示，过去一段时间涨幅靠前的资产，在随后一段时间继续跑赢的概率 > 50%。

### 1.2 为什么动量有效

三层机制：

| 层级 | 机制 | 说明 |
|------|------|------|
| 信息扩散 | 好消息需要时间传播 | 不是所有人同时知道一个币要上币安——信息从 insider → 聪明钱 → KOL → 散户，逐层扩散 |
| 行为偏差 | 反应不足 + 羊群效应 | 人对新信息的第一反应通常不足（"再看看"），等确认了才追——追的过程就是动量 |
| 市场结构 | 趋势跟踪者的自我实现 | 趋势策略越多，动量越强——因为大家都在等趋势确认后入场 |

### 1.3 单层动量 vs 双层动量

单层动量（只看一个时间窗口）的问题：

```
MOM(10) > 0 → 做多
```

过去 10 天涨了，但过去 30 天在跌——这只是一次短期反弹，不是趋势反转。单层动量会被反弹骗进去。

双层动量的逻辑：

```
MOM(10) > 0 AND MOM(30) > 0 → 做多
```

短期动量和长期动量方向一致 = 反弹不是反弹，是趋势的延续。**双层过滤避免单一时间框架的假信号。**

这是我们的动量策略和普通动量策略的关键区别。

### 1.4 为什么我们的动量策略在回测中失败了

BTC 日线动量策略：夏普 -0.82，总亏损 -34.8%，62 笔交易。

根因分解：

| 问题 | 数据 | 根因 |
|------|------|------|
| 交易太多 | 62 笔/两年（趋势跟踪仅 14 笔） | 动量信号频繁翻转，入场→止损→再入场 |
| 成本吞噬 | 总成本 12.4%，占总利润 55% | 高频 = 高成本，天然劣势 |
| 止损被频繁触发 | 回撤 -52.2% | ATR(2.0) 止损在动量策略上太紧 |
| 信号出场不及时 | 动量衰减后才平仓 | `fast < 0 or slow < 0` 的条件太滞后 |

**核心矛盾**：动量策略需要一定的交易频率来捕捉中短期趋势，但每一笔交易都要交 0.2% 的「税」。频率 × 成本 = 不可逾越的摩擦。

### 1.5 动量策略的改进方向

如果未来要重新启用动量策略：

1. **加信号强度过滤**：动量不仅要 > 0，还要 > 某个阈值（比如 5%）才入场
2. **放宽止损**：动量策略的波动容忍度应该比趋势策略更大
3. **降频**：只在动量加速（二阶导数为正）时入场，过滤横盘动量
4. **与趋势策略信号对齐**：只有当趋势策略也同向持仓时，动量策略才开仓

## 二、资金费率套利

### 2.1 资金费率是什么

永续合约没有到期日。为了让合约价格锚定现货价格，交易所引入了**资金费率机制**：

```
合约价格 > 现货价格 → 多头付钱给空头（正费率）
合约价格 < 现货价格 → 空头付钱给多头（负费率）
```

每 8 小时结算一次。年化费率通常在 -30% ~ +80% 之间波动。

### 2.2 套利原理

```
操作：现货买入 BTC + 永续合约做空等量 BTC
结果：价格涨跌和你无关（现货赚的 = 合约亏的，互相抵消）
收益：每 8 小时收取一次资金费率（多头付给你的）
```

**这是市场中性策略**——你的收益不依赖价格涨跌，只依赖资金费率。不管 BTC 涨到 10 万还是跌到 1 万，你的 PnL 只和费率有关。

### 2.3 为什么是基准策略

路线图写得很清楚：**「其他策略必须跑赢它才有存在的意义」**。

原因：

| 维度 | 资金费率套利 | 方向性策略（趋势/动量） |
|------|------------|---------------------|
| 方向风险 | 无（对冲掉了） | 有（赌方向） |
| 收益来源 | 结构性（市场机制） | 预测性（猜方向） |
| 可预测性 | 高（费率数据公开透明） | 低（价格不可预测） |
| 逻辑复杂度 | 极简 | 较复杂 |
| 失败模式 | 费率反转变负 | 方向判断错误 |

**如果你费尽心思设计的方向性策略跑不过一个机械化的费率套利，那你的 edge 在哪里？**

### 2.4 我们的回测验证

BTC 日线费率套利：+27.8%，夏普 0.44，胜率 80%，15 笔交易。

三品种表现：

| 币种 | 收益 | 夏普 | 胜率 | 平均入场费率 |
|------|------|------|------|------------|
| BTC | +27.8% | 0.44 | 80.0% | 25.7% 年化 |
| ETH | +36.3% | 0.51 | 80.0% | — |
| SOL | +40.5% | 0.54 | 94.1% | — |

三品种夏普全部为正。胜率 80-94%。**这就是基准策略应该有的样子：稳，简单，可复制。**

### 2.5 费率套利的真正风险

回测中用模拟数据看起来很美好，但实盘有三个真实风险：

| 风险 | 说明 | 发生概率 | 防法 |
|------|------|---------|------|
| 费率反转 | 正费率变负，你从收钱变成付钱 | 中（牛市转熊时） | 设 exit_funding_rate 阈值，费率 < 5% 就平仓 |
| 基差风险 | 现货和合约价差变动，对冲不完美 | 低 | 开仓时基差 < 0.5% 才入场 |
| 交易所风险 | 交易所跑路/爆雷 | 低但致命 | 盈利定期提现，不放太多资金在交易所 |

### 2.6 费率套利的收益上限

费率套利的年化收益天花板大概是 30-40%。为什么不会更高？

- 费率 > 50% 年化通常出现在极端 FOMO 行情中，不可持续
- 高费率吸引更多套利者入场 → 合约卖压增加 → 费率自然下降
- 这是一个自我修正的市场机制

**所以费率套利是稳稳的 10-30% 年化，但不是暴富工具。它保护你的本金，趋势策略负责进攻，两者角色不同。**

## 三、动量因子 vs 资金费率套利：一个更深层的对比

### 3.1 两种赚钱逻辑的本质区别

| | 动量因子 | 资金费率套利 |
|------|---------|------------|
| 赚谁的钱 | 反应慢的交易者（信息劣势方） | 做多永续合约的人（他们付你费率） |
| 为什么能持续赚钱 | 信息扩散需要时间，行为偏差不会消失 | 永续合约的结构性设计决定了多头常态性付钱 |
| 什么时候失效 | 市场完全有效的那一天（永远不会） | 费率机制被废除的那一天（不太可能） |
| 可扩展性 | 有限（太多人用会减弱动量效应） | 有限（太多套利者 → 费率下降） |

### 3.2 为什么动量策略跑输了费率套利

三策略中动量策略是唯一三品种全负夏普的（0% 正夏普率）。根本原因：

**动量策略的交易频率（62 笔/两年）是中频策略，但我们的成本结构（0.2%/往返）是给低频策略设计的。**

成本和频率的错配是动量策略失败的根因。如果每笔交易成本能降到 0.05%，动量策略可能就活了。但在当前交易所费率结构下（现货 0.1%），这个假设不成立。
