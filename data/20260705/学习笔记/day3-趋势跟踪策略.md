# Day 3 学习笔记 — 趋势跟踪策略

## 一、趋势策略的赚钱逻辑

### 1.1 核心命题

> 趋势跟踪之所以能赚钱，不是因为能预测趋势，而是因为趋势一旦形成，延续的时间比你想象的长，幅度比你想象的大。

这不是玄学，背后有三层逻辑：

| 层级 | 机制 | 为什么趋势会延续 |
|------|------|-----------------|
| 行为金融 | 锚定效应 + FOMO | 人看到涨了就想追，跌了就想割——不是理性决策，是情绪驱动 |
| 市场结构 | 止损盘踩踏 + 杠杆爆仓 | 空头的止损买单 = 多头的燃料；多头的止损卖单 = 空头的燃料 |
| 叙事驱动 | 故事需要时间传播 | "BTC 要上 10 万"这个故事从早期 adopters 传到散户需要几周甚至几个月 |

### 1.2 趋势跟踪的数学特征

```
胜率：30-40%（大部分交易小亏）
盈亏比：2:1 ~ 4:1（少数交易大赚）
期望值 = 胜率 × 平均盈利 - (1-胜率) × 平均亏损
        = 0.35 × 3 - 0.65 × 1 = 0.4 → 正期望！
```

**关键认知**：趋势跟踪是一种「靠少数大赚覆盖多数小亏」的策略。你不能指望每笔都赚钱——你必须接受连续亏损 6-7 次是正常的。

### 1.3 我们的实际验证

BTC 日线 EMA(10/40) 趋势跟踪：
- 胜率 33% ✅（和理论一致）
- 盈亏比 2.08 ✅（和理论一致）
- 9 笔交易，3 笔盈利覆盖 6 笔亏损还有剩

**这恰好验证了趋势跟踪的数学特征**——不是策略设计得好，是这个策略类型的自然属性。

## 二、均线系统

### 2.1 EMA vs SMA

| | EMA（指数移动平均） | SMA（简单移动平均） |
|------|-------------------|---------------------|
| 权重 | 近期价格权重更高 | 所有价格等权重 |
| 灵敏度 | 更快响应趋势变化 | 滞后更大 |
| 假信号 | 更多 | 更少 |
| 适用 | 中短期趋势（我们的日线 10/40） | 长期趋势 |

我们选 EMA 的原因：币圈趋势变化快，SMA 等你确认趋势的时候行情已经走了一半。

### 2.2 双均线交叉的本质

```
快线（10 日）穿越慢线（40 日）= 短期共识追上长期共识
```

- 金叉（快线上穿慢线）：短期看涨力量压倒长期均值 → 做多
- 死叉（快线下穿慢线）：短期看跌力量压倒长期均值 → 做空

**双均线不是预测工具，是确认工具**。它不预测趋势，它确认趋势已经形成。等你看到金叉，趋势已经走了一段了——这段「已经走了的」是信号的成本，后面「还会继续走的」是利润。

### 2.3 参数选择的深层逻辑

快线 10 / 慢线 40（日线）意味着：
- 快线追踪 ~2 周的价格共识
- 慢线追踪 ~2 个月的价格共识
- 交叉点 = 短期共识 vs 中期共识的分歧 → 收敛

如果快线太短（5 日）→ 噪音太多，假信号频繁
如果慢线太长（60 日）→ 滞后严重，趋势结束了还没确认

## 三、突破策略

### 3.1 Donchian Channel 的逻辑

```
上轨 = 过去 N 日最高价
下轨 = 过去 N 日最低价
中轨 = (上轨 + 下轨) / 2

价格突破上轨 → 市场选择了向上突破 → 做多
价格跌破下轨 → 市场选择了向下突破 → 做空
价格回归中轨 → 方向性减弱 → 平仓
```

### 3.2 为什么突破策略在回测中失败了

我们的实际数据：通道突破三品种平均夏普 -0.19，EMA 交叉 +0.41。

根因分析：

| 问题 | 数据 | 解释 |
|------|------|------|
| 假突破太多 | 27 笔/两年（EMA 仅 9 笔） | 突破后经常立即反转 |
| 胜率太低 | 29.6%（EMA 33.3%） | 每 3 次突破只有 1 次是真的 |
| 回撤失控 | BTC -41.5%, SOL -61.9% | 假突破 + 止损 = 反复被割 |

**突破策略的致命缺陷**：它假设「突破 = 趋势启动」。但在币圈，大量突破是假突破——大户故意推穿关键价位触发止损盘和 FOMO 追涨，然后反手出货。

EMA 交叉之所以更稳，是因为它要求**价格和均线的相对位置持续一段时间**才确认——天然过滤了瞬时假突破。

### 3.3 突破策略有价值的部分

通道突破和 EMA 交叉的信号重叠度只有 3%——说明它们捕捉的是不同类型的趋势启动点。突破策略在**波动率扩张**的转折点上有独特价值（EMA 还没反应过来的时候，价格已经突破了）。

改进方向（如果需要）：
- 加入成交量确认（突破 + 放量 = 真突破概率更高）
- ATR 噪音过滤（突破幅度必须 > 1×ATR 才算有效突破）
- 只在 ADX > 25 的环境下允许入场

## 四、趋势的叙事基础

### 4.1 币圈趋势为什么特别强

传统股市：趋势延续性中等，因为估值有锚（PE、PB）
币圈：趋势延续性极强，因为没有估值锚——一个币「应该值多少钱」完全由叙事决定

叙事驱动的趋势特征：
1. **启动慢**：早期只有少数人相信这个故事
2. **加速快**：故事传播 → FOMO 入场 → 价格自我实现
3. **崩塌猛**：叙事被证伪 → 没有估值底 → 自由落体

我们的做空收益验证了这一点：BTC 做空 +14.7%（7 笔），SOL 做空 +67.5%（6 笔）。下跌趋势的暴力程度不亚于上涨。

### 4.2 「这个故事还能讲多久」

趋势策略持仓期间，你不应该看 K 线，你应该看叙事：
- 这个币的故事还在传播吗？（Google Trends / 社交提及量在上升）
- 这个故事的核心假设被证伪了吗？（监管 / 技术漏洞 / 团队跑路）
- 有新的替代叙事出现吗？（竞品抢走了注意力）

**叙事活着 = 趋势活着。叙事死亡 = 趋势死亡。**

## 五、趋势策略的死穴：震荡市

### 5.1 为什么震荡市是趋势策略的天敌

震荡市 = 价格在一个区间里来回走。每一次「突破」都是假的，每一次「趋势确认」都是区间边缘。

趋势策略在震荡市的行为：
```
价格涨到区间上沿 → EMA 金叉 → 做多 → 价格跌回区间 → 止损
价格跌到区间下沿 → EMA 死叉 → 做空 → 价格涨回区间 → 止损
反复被割。
```

### 5.2 怎么识别震荡市

| 指标 | 震荡市特征 | 趋势市特征 |
|------|-----------|-----------|
| ADX | < 20 | > 25 |
| 布林带宽度 | 收窄 | 扩张 |
| 均线关系 | 快慢线反复交叉 | 快线持续在慢线一侧 |
| 价格行为 | 在一个区间内反复 | 持续创出新高/新低 |

### 5.3 我们的解决方案

Day 11（市场环境分类器）会解决这个问题：震荡市自动降仓到 30%，趋势市全仓。不是改策略，是管仓位。
